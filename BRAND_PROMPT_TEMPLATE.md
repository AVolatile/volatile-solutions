## Brand Snapshot

Volatile Solutions should feel like a modern, trust-first digital services brand for small and mid-sized businesses.
The design direction is clean, premium, and credible, with a slight SaaS/product feel rather than a playful agency look.

The visual system uses two layers:
- Identity layer: teal + charcoal from the logo.
- Web UI layer: navy/blue gradients, teal accents, soft white surfaces, and restrained gray typography.

## Color System

Primary UI colors:
- `#13547A` - primary brand blue
- `#1A6FA0` - secondary blue
- `#0E2F48` - dark hover blue
- `#F0F8FF` - light section background
- `#E3E9F2` - border color
- `#717275` - muted body text
- `#FFFFFF` - white
- `#000000` - dark text base

Expanded brand accents used in hero/nav/footer treatments:
- `#09101A` - deep navy hero/footer base
- `#101C2B` - layered navy
- `#16283B` - navy gradient end
- `#1EA9A0` - teal accent / CTA energy
- `#74DECF` - lighter teal highlight

Logo direction:
- Teal mark with charcoal/gray supporting text
- Clean white background for identity usage

## Typography

Use:
- Headings: `Montserrat`, semibold to bold
- Body copy: `Open Sans`, regular to semibold

Type style:
- Headlines are clear, bold, and compact.
- Nav and small labels use uppercase with generous letter spacing.
- Body copy is readable, calm, and businesslike.
- Avoid decorative, script, or overly futuristic fonts.

## Design DNA

The design should feel:
- polished
- trustworthy
- modern
- calm
- conversion-focused
- premium without feeling flashy

The interface should use:
- dark gradient hero sections
- light content sections
- rounded cards with soft borders
- pill buttons and pill nav items
- subtle glassmorphism in high-impact areas only
- soft shadows, not heavy neumorphism
- restrained motion and hover lift
- clean spacing with obvious section separation

## Layout Rules

Follow these structural patterns:
- Hero first: dark gradient background, strong headline, short supporting copy, clear CTA, proof points.
- Alternate dark and light sections for rhythm.
- Use cards for services, process steps, portfolio previews, and pricing.
- Keep content width controlled and readable.
- Use badges, eyebrows, and small uppercase labels to organize sections.
- Prioritize clarity over density.

## Copy Tone

The brand voice should sound:
- direct
- credible
- clear
- practical
- low-hype
- business-oriented

Prefer copy about:
- trust
- clarity
- lead generation
- professionalism
- simplicity
- low-friction process
- visible business outcomes

Avoid copy that feels:
- trendy
- gimmicky
- over-promising
- overly technical without context
- startup-jargon heavy

## Prompt Template

Use this when generating future pages, sections, ads, or UI concepts:

```text
Create a [page/section/design] for Volatile Solutions that matches the existing brand.

Brand personality:
- modern, trustworthy, premium, calm, conversion-focused
- built for small and mid-sized businesses
- more polished and credible than flashy or playful

Visual direction:
- dark navy gradient hero backgrounds using tones like #09101A, #101C2B, #16283B
- primary brand blue #13547A with supporting blue #1A6FA0
- teal accent moments using #1EA9A0 and lighter teal highlights like #74DECF
- white and very light blue surfaces like #F0F8FF
- subtle borders in #E3E9F2
- body text in muted gray like #717275

Typography:
- headings in Montserrat, semibold to bold
- body copy in Open Sans
- small labels and nav items in uppercase with tracking

UI style:
- rounded cards
- pill buttons
- soft shadows
- light glassmorphism only where it adds polish
- clean spacing and strong section hierarchy
- dark hero, light content sections, premium footer

Tone of copy:
- clear, direct, confident, business-focused
- emphasize trust, clarity, professionalism, and lead generation
- avoid hype, slang, or playful startup language

Do not:
- use random bright colors
- make it look childish, trendy, or crypto-themed
- use generic startup gradients like purple/pink
- overcomplicate the layout
- use harsh contrast everywhere

The final result should feel like a premium web agency / digital solutions brand that helps businesses look credible and convert more visitors into leads.
```

## Short Prompt Version

```text
Match the existing Volatile Solutions branding: modern trust-first B2B web design, Montserrat headlines, Open Sans body text, navy-to-blue gradient heroes, teal accents, white/light-blue surfaces, rounded cards, pill CTAs, soft shadows, and clean conversion-focused layouts. Keep the tone direct, credible, and business-oriented.
```

## Visual Do / Don't

Do:
- use navy, blue, teal, white, and restrained gray
- make the design feel structured and premium
- use gradients intentionally, mostly in hero/footer/featured areas
- keep icons simple and professional
- highlight CTAs with teal or white on dark backgrounds

Don't:
- use purple as a primary color
- use cartoonish illustrations
- use brutalist or ultra-experimental layouts
- make everything dark; the brand needs contrast through light sections
- add excessive glow, blur, or animation
